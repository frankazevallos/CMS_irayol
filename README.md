## 🔧 Installation 

Install my-project with npm

```bash 
  git clone https://github.com/jesusave/irayol.git

  cd irayol

  composer install

  php artisan key:generate

  php artisan migrate --seed

  php artisan storage:link

  http://127.0.0.1:8000

  User: admin@admin.com or admin
  Pass: 123456
```
